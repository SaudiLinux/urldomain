## نظرة عامة
أداة بايثون قوية وفعالة لفحص الدومينات واستخراج النطاقات الفرعية والروابط المخفية، بالإضافة إلى فحص واكتشاف الثغرات الأمنية الشائعة وتصنيفها حسب درجة الخطورة.

## الميزات الرئيسية

### 1. استخراج النطاقات الفرعية (Subdomain Enumeration)
- فحص أكثر من 150 نطاق فرعي شائع
- استخراج سجلات DNS المختلفة (A, AAAA, MX, NS, TXT, SOA, CNAME, SRV, CAA)
- البحث في قاعدة بيانات crt.sh عن شهادات SSL لاستخراج نطاقات إضافية
- دعم الفحص المتعدد الخيوط للسرعة

### 2. الزحف واستخراج الروابط
- زحف عميق على الموقع لاستخراج جميع الروابط
- اكتشاف الروابط المخفية والمسارات الحساسة
- استخراج نقاط نهاية API من ملفات الجافاسكريبت
- تحليل ملفات JavaScript للعثور على نقاط الوصول المخفية

### 3. فحص الثغرات الأمنية
تقوم الأداة باكتشاف وتصنيف الثغرات التالية:
- **SQL Injection** (حرجة) - حقن استعلامات SQL
- **Command Injection** (حرجة) - تنفيذ أوامر النظام
- **XSS (Cross-Site Scripting)** (عالية) - حقن كود جافاسكريبت
- **LFI/RFI** (عالية/حرجة) - تضمين الملفات المحلية والبعيدة
- **SSRF** (عالية) - تزوير الطلبات من جهة الخادم
- **XXE Injection** (عالية) - استغلال كيانات XML الخارجية
- **CORS Misconfiguration** (متوسطة) - إعدادات CORS خاطئة
- **Missing CSRF Protection** (متوسطة) - عدم وجود حماية CSRF
- **Open Redirect** (متوسطة) - إعادة توجيه مفتوحة

### 4. فحص إضافي
- فحص المنافذ المفتوحة على الخادم
- تحليل شهادات SSL/TLS والكشف عن المشكلات الأمنية
- اكتشاف جدران الحماية (WAF)
- إنشاء تقارير مفصلة بتنسيق JSON

## الاستخدام

### الفحص الكامل لدومين
```bash
python domain_security_scanner.py example.com
```

### تحديد عدد الخيوط
```bash
python domain_security_scanner.py example.com --threads 20
```

### حفظ التقرير في ملف
```bash
python domain_security_scanner.py example.com --output report.json
```

### أوامر متخصصة
```bash
# استخراج النطاقات الفرعية فقط
python domain_security_scanner.py example.com --subdomains-only

# الزحف واستخراج الروابط فقط
python domain_security_scanner.py example.com --crawl-only

# فحص الثغرات فقط
python domain_security_scanner.py example.com --vuln-scan-only
```

## المخرجات
تعرض الأداة أثناء الفحص:
1. النطاقات الفرعية المكتشفة
2. الروابط المخفية ونقاط API
3. المنافذ المفتوحة
4. الثغرات المكتشفة مع تفاصيلها ودرجة خطورتها
5. ملخص نهائي بجميع النتائج

## تصنيف درجات الخطورة
- **حرجة**: ثغرات خطيرة جداً تسمح بالوصول الكامل للنظام
- **عالية**: ثغرات تسمح باختراق البيانات أو الوصول غير المصرح به
- **متوسطة**: ثغرات تتطلب ظروف معينة للاستغلال
